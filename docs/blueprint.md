# **App Name**: PresenciaTech

## Core Features:

- Secure User Authentication: Robust login and registration forms for administrators and instructors, ensuring secure access and role-based permissions.
- Curriculum Management: Admin dashboard features for comprehensive CRUD (Create, Read, Update, Delete) operations on Study Programs and Course Units.
- Instructor Management: Admin functionalities to manage instructor profiles, assign them to specific Course Units, and update their details.
- Bulk Data Import Tool: A 'dropzone' UI for administrators to upload Excel files for efficient mass import of Study Programs, Course Units, and Instructor data.
- Instructor Course Overview: A 'Cards' style dashboard for instructors to quickly view all assigned Course Units, clearly indicating their associated Study Program.
- Mobile-Optimized Attendance Interface: A clean, responsive table interface optimized for tablets and mobile phones, displaying student avatars, names, and quick-action buttons for 'Presente', 'Tarde', 'Falta', and 'Justificado' statuses.
- AI Attendance Insights Tool: An AI-powered tool that analyzes historical attendance data to generate summaries, identify trends, and highlight patterns for specific students or course units, aiding administrators and instructors in student support.

## Style Guidelines:

- Primary color: A deep, professional institutional blue (#2261CB), evoking trust and stability in an educational setting.
- Background color: A very light blue-gray (#EBF1F9), providing a clean, subtle foundation that enhances readability and visual calm.
- Accent color: A soft, vibrant lavender-purple (#8F66E5) to draw attention to interactive elements and highlight key information with a modern flair.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clear, and highly readable design, suitable for a professional system with diverse content types.
- Utilize a consistent set of modern, minimalist line-art icons. For student lists, use circular avatars to personalize the attendance interface.
- Administrator dashboard will feature a left-hand sidebar for primary navigation, enabling quick access to different management sections. Instructor dashboards will use a card-based layout for clear course overviews. All critical interfaces, especially the attendance tracker, will be designed for responsiveness across desktop, tablet, and mobile devices.
- Implement subtle, fast transitions for navigation between sections and feedback on button presses or form submissions, ensuring a fluid user experience without distractions.
